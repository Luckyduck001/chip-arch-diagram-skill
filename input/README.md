# 输入文件说明

## 你需要准备两个文件

### 1. 架构图描述输入（.md 文件）

复制 `架构图描述输入模板.md`，按需修改各章节。

**连接关系怎么写——最重要！**

| 连接类型 | 写法 | 画法 |
|---------|------|------|
| 同行水平 | `A → B → C` | AI 自动画水平直线 |
| 同列垂直 | `A → B` | AI 自动画垂直直线 |
| 内部↔外围 | `PIO → MOD` | AI 自动画直线 |
| **不同行且不同列** | `【L】A（边）→ B（边）` | **你必须指定出入边**，AI 画 L 形 |

边的写法：`顶` `底` `左` `右`。

```
PIO_A → MOD_A → MOD_B → MOD_C
【L】MOD_H（右） → MOD_C（底）
【L】MOD_K（右） → MOD_I（左）
```

### 2. Excel 网格表（.xlsx 文件）

复制 `架构网格图模板.xlsx`，在 `网格布局` sheet 中把示例模块名替换成你自己的。

规则：
- 每个格子填模块简称，空单元格留空
- 跨行/跨列模块每格都填（同名填充）
- 外围模块放四边：最左列(c=0)、最右列、最上行(r=0)、最下行

### 3. 发给 AI

> "生成架构图。输入文件在 `input/我的芯片.md`，Excel 在 `input/我的网格.xlsx`"

> 环境要求：Python + openpyxl（`pip install openpyxl`）
