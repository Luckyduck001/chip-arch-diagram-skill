# 芯片架构图生成器

把 Excel 网格表变成芯片架构图 SVG。**不需要写代码。**

## 5 分钟上手

### 1. 准备两个文件

| 文件 | 怎么做 |
|------|--------|
| **架构图描述** (.md) | 复制 `input/架构图描述输入模板.md`，填好模块名、连接、颜色、尺寸、父框 |
| **Excel 网格表** (.xlsx) | 在 Excel 里画格子，每个格子填模块名，空单元格留空 |

详细说明见 `input/README.md`。

### 2. 发一句话给 AI

> "生成架构图。输入：`你的描述文件.md`，Excel：`你的网格.xlsx`"

AI 自动调用 `generator.py`，一步完成解析→计算→路由→HTML→自检。

> **环境要求**：Python + openpyxl（`pip install openpyxl`）

或者直接命令行：
```bash
python generator.py input/描述.md input/网格.xlsx
```

### 3. 拿到结果

AI 生成 `output/架构图.html`，浏览器打开就能看。

## 核心设计：你管路由，AI 管计算

| 连接类型 | 谁负责 | 怎么写 |
|---------|--------|--------|
| 同行水平 | AI 自动 | `A → B` |
| 同列垂直 | AI 自动 | `A → B` |
| 内部↔外围 | AI 自动 | `PIO → MOD` |
| **不同行且不同列** | **你指定边** | `【L】A（右）→ B（底）` |

AI 擅长坐标计算和 HTML 生成，但判断"从哪个边出去"容易出错。
你把边指定好，剩下的全自动。

## 文件夹结构

```
chip-arch-diagram/
├── SKILL.md                       # AI 执行指令（执行驱动 + 关键公式摘要）
├── generator.py                   # 核心库（封装全部 11 个已知陷阱）
├── README.md                      # 本文件
├── rules/
│   ├── 通用布局规则.md             # 完整布局算法（坐标、父框、颜色、标签）
│   ├── 通用连线规则.md             # 完整路由算法（Phase 1/2/3、k值、交叉检测）
│   └── 通用检查清单.md             # 完整自检清单（A-I 共 30+ 项）
└── input/
    ├── README.md                  # 输入文件怎么写
    ├── 架构图描述输入模板.md       # 复制这个开始填
    └── 架构网格图模板.xlsx         # 复制这个画网格
```

## 换一个芯片项目

改两个文件，其他不动：
1. 新的 `架构图描述输入.md`
2. 新的 Excel 网格表 `.xlsx`
